# IMDb Movie Rating Scraper

This project collects movie details from the IMDb Top 250 Movies page using Python and Selenium.  
It automatically extracts the movie *rank, title, year, and rating* and saves the results in a *CSV file*.

## Purpose
To automate data collection from IMDb instead of doing it manually, and to learn web scraping using Selenium.

## Features
- Scrapes IMDb Top 250 Movies
- Extracts Rank, Title, Year, Rating
- Saves data to imdb_top250.csv
- Simple and easy to run

## Technologies Used
- Python
- Selenium
- WebDriver Manager
- Pandas
